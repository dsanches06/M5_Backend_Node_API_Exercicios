# Users

curl http://localhost:3000/users
curl -X POST http://localhost:3000/users -H "Content-Type: application/json" -d '{"name":"João","email":"joao@example.com"}'
curl -X PUT http://localhost:3000/users -H "Content-Type: application/json" -d '{"id":1,"name":"Alice Souza"}'
curl -X DELETE http://localhost:3000/users -H "Content-Type: application/json" -d '{"id":2}'

💡 Exercícios de Backend — APIs HTTP (Mini ClickUp Use Cases Diferentes)

📌 Regras gerais:
- Criar servidor Node.js com módulo http
- Usar JSON nas respostas

🧑‍🏫 Exercícios Guiados (Em Aula)

📝 Exercício 1 — Criar lista de utilizadores

Cria um array users com objetos: id, name, email.

Endpoint: GET /users — retorna todos os utilizadores.

📝 Exercício 2 — Criar novo utilizador

Endpoint: POST /users

Body JSON: 

{
  "name": "João Silva",
  "email": "joao@example.com"
}
Dica: gerar id automático e adicionar ao array.

📝 Exercício 3 — Atualizar utilizador

Endpoint: PUT /users

Body JSON: 

{
  "id": 1,
  "name": "João Souza"
}
Dica: encontrar pelo id e atualizar nome.

📝 Exercício 4 — Remover utilizador

Endpoint: DELETE /users

Body JSON: 

{
  "id": 1
}
Dica: usar filter() para remover do array.

📝 Exercício 5 — Criar tags

Cria um array tags com objetos: id, name.

Endpoint: GET /tags — retorna todas as tags.

📝 Exercício 6 — Adicionar tag a tarefa

Endpoint: POST /task-tags

Body JSON: 

{
  "taskId": 2,
  "tagId": 1
}
Dica: criar array taskTags para relacionar tarefas e tags.

📝 Exercício 7 — Criar comentários

Cria um array comments com objetos: id, taskId, userId, content.

Endpoint: GET /comments?taskId=1 — retorna comentários de uma tarefa específica.

📝 Exercício 8 — Adicionar comentário

Endpoint: POST /comments

Body JSON: 

{
  "taskId": 1,
  "userId": 2,
  "content": "Comentário de teste"
}
Dica: adicionar ao array comments com id automático.

📝 Exercício 9 — Marcar comentário como resolvido

Endpoint: PUT /comments

Body JSON: 

{
  "id": 3,
  "resolved": true
}
Dica: adicionar propriedade resolved: false inicialmente, atualizar para true.

📝 Exercício 10 — Listar notificações

Cria um array notifications com objetos: id, userId, message, read.

Endpoint: GET /notifications?userId=1 — retorna notificações de um utilizador específico.

🎯 Objetivos da Aula

✔ Aprender a criar endpoints REST para diferentes use cases
✔ Manipular arrays e relacionamentos simples entre dados
✔ Ler query parameters e body JSON
✔ Implementar métodos HTTP (GET, POST, PUT, DELETE)
✔ Preparar para trabalhar com APIs reais tipo ClickUp

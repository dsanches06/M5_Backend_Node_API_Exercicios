💡 Exercícios Autónomos — Mini ClickUp APIs

📌 Regras gerais:
- Usar Node.js e módulo http
- Trabalhar com arrays JSON e múltiplos relacionamentos

📝 Exercício 1 — Atribuir múltiplos utilizadores a uma tarefa

Cria um array taskUsers com objetos: taskId e userId.

Endpoint: POST /task-users
Body JSON: 

{
  "taskId": 2,
  "userId": 1
}

📝 Exercício 2 — Notificação automática ao adicionar comentário

Quando um comentário é criado, adiciona automaticamente uma notificação para o userId do comentário.

Dica:
- No endpoint POST /comments, após adicionar o comentário, cria um objeto notifications com message informando sobre o comentário.

📝 Exercício 3 — Listar tarefas concluídas e pendentes

Endpoint: GET /tasks?done=true ou done=false

Dica:
- Ler query parameter done
- Filtrar o array tasks usando filter() e retornar JSON apenas com tarefas correspondentes.

📝 Exercício 4 — Marcar tarefa como favorita

Adicionar propriedade favorite a tarefas e criar endpoint PUT /tasks/favorite

Body JSON: 

{
  "taskId": 1,
  "favorite": true
}

Dica: atualizar a tarefa correspondente no array tasks e retornar o objeto atualizado.

🎯 Objetivos dos Exercícios Avançados

✔ Praticar relações muitos-para-muitos (task-users)
✔ Aprender a criar notificações automáticas
✔ Trabalhar com filtros e query parameters avançados
✔ Consolidar lógica de API REST real
